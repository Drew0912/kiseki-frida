# Kiseki Frida
Frida script to extend functionality of Kiseki/Trails games and enable dynamic debugging.

Updated from https://github.com/Drew0912/ED8Frida which was based on https://github.com/Ouroboros/Falcom/tree/master/ED8/ed83 by [Ouroboros](https://github.com/Ouroboros) to enable more features of Frida.

Based on Frida version 16.4.10. Later versions have compatability issues related to Windows, e.g does not work on Windows 10 and system errors hooking WinAPI.
