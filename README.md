# Kiseki Frida
Frida script to extend functionality of Kiseki/Trails games and enable dynamic debugging.

Updated from https://github.com/Drew0912/ED8Frida which was based on https://github.com/Ouroboros/Falcom/tree/master/ED8/ed83 by [Ouroboros](https://github.com/Ouroboros) to support newer versions of Frida and enable more features of Frida.
